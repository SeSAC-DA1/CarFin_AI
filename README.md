## 🔍 숨은 니즈 발굴 시스템

### **대화형 프로파일링으로 진짜 니즈 발굴**

#### **❌ 직접적 질문 vs ✅ 간접적 추론**

```yaml
기존 방식:
  질문: "어떤 차 원하세요?"
  답변: "경제적인 차요"
  결과: 뻔한 추천

CarFin AI 방식:
  질문: "평일과 주말에 주로 어떤 활동을 하시나요?"
  답변: "평일엔 출퇴근, 주말엔 친구들과 드라이브나 골프"
  AI분석: 스타일링 중시 + 트렁크 공간 필요 + 주행성능 관심
  결과: 개인 맞춤 완벽 추천
```

#### **실제 니즈 발굴 과정**

**1단계: 라이프스타일 탐지**
- 사용 패턴, 활동, 환경 분석으로 숨은 니즈 추론
- "골프" 언급 → 트렁크 공간 + 이미지 중시 자동 인식

**2단계: 협업 필터링 활용**
- "28세 IT회사 직장인, 강남 거주" → 유사 프로필 분석
- "비슷한 분들의 83%가 스타일링과 기술옵션을 중시했습니다"

**3단계: NLP 감정 분석**
- "연비는 그냥 평범해도 괜찮아요" → 경제성 낮은 우선순위
- "디자인이 예뻐야 해요" → 외관 높은 우선순위

## 🤖 3-Agent AI 분석 시스템

### **"니 취향만 말해, 나머지는 내가 다 해줄게" - 최적화된 3명의 AI 전문가**

#### **🎯 Agent 1: 컨시어지 매니저**
```yaml
전문분야: 전체 상담 프로세스 관리 + 사용자 경험 최적화
성격: 친근하고 효율적인 개인 비서
역할:
  - 사용자 첫 접점 및 니즈 초기 파악
  - N번째 질문 대응 및 상담 흐름 관리
  - 다른 에이전트들과의 협업 조율
  - 최종 결과 종합 및 명확한 프레젠테이션
```

#### **🔍 Agent 2: 니즈 분석 전문가**
```yaml
전문분야: 숨은 니즈 발굴 + 개인화 프로파일링 (CarFin AI 핵심)
성격: 예리하고 통찰력 있는 심리 분석가
역할:
  - 대화에서 숨은 니즈 추론 ("골프도 치는데..." → 트렁크 공간)
  - 협업 필터링으로 유사 고객 패턴 분석
  - 라이프스타일 기반 개인화 프로파일 생성
  - "당신도 몰랐던 진짜 필요한 기능" 발굴
```

#### **📊 Agent 3: 데이터 분석 전문가**
```yaml
전문분야: 올인원 데이터 분석 + 매물 추천 (CarFin AI 핵심 엔진)
성격: 정확하고 종합적인 데이터 사이언티스트
역할:
  - 111,842+ 매물 DB 실시간 검색 & 필터링
  - TCO 계산 + EDA 기반 감가상각률 & 잔가 예측
  - 금융상품 최적 매칭 (할부/리스/일시불)
  - 리스크 분석 & 최종 추천 생성
  - "숨은 보석" 매물 발굴
```

## 🔧 3-Agent 시스템 기술 아키텍처 분석

### **🎯 Agent 1: 컨시어지 매니저 - 상담 프로세스 관리**

```yaml
핵심_기술_스택:
  LLM: Google Gemini 2.5 Flash + A2A Protocol
  역할: "상담 플로우 제어 + 사용자 경험 최적화"

구현_기능:
  - 대화 컨텍스트 관리 및 상담 단계 추적
  - 사용자 질문 의도 분석 및 적절한 에이전트 라우팅
  - 3-Agent 협업 결과 통합 및 최종 프레젠테이션
  - N번째 질문 대응 및 만족도 모니터링

실용적_기술_구현:
  - Session Management: Redis (검증된 세션 저장소)
  - Intent Classification: 키워드 기반 규칙 + 간단한 패턴 매칭
  - Response Orchestration: 조건문 기반 에이전트 라우팅
  - Context Tracking: JSON 형태로 대화 히스토리 관리
```

### **🔍 Agent 2: 니즈 분석 전문가 - 숨은 니즈 발굴 (CarFin AI 핵심)**

```yaml
핵심_기술_스택:
  LLM: Google Gemini 2.5 Flash + A2A Protocol + 한국어 특화 NLP
  역할: "대화 분석 + 개인화 프로파일링"

구현_기능:
  - 대화에서 명시적/암시적 니즈 추출
  - 한국어 감정 분석 및 우선순위 가중치 계산
  - 협업 필터링으로 유사 고객 패턴 매칭
  - 라이프스타일 기반 숨은 요구사항 예측

실용적_기술_구현:
  - 한국어_감정분석: KoBERT 한국어 특화 모델
  - 키워드_추출: KoNLPy (Okt) 형태소 분석
  - 협업_필터링: scikit-learn Cosine Similarity (검증된 알고리즘)
  - 콜드_스타트_해결: 연령대/지역/직업 기반 인구통계학적 유사성
  - 니즈_가중치: TF-IDF 기반 키워드 중요도 계산
  - 패턴_매칭: 규칙 기반 + 빈도수 분석
```

### **📊 Agent 3: 데이터 분석 전문가 - 올인원 분석 엔진**

```yaml
핵심_기술_스택:
  LLM: Google Gemini 2.5 Flash + A2A Protocol + Python 백엔드
  역할: "111,842+ 매물 분석 + 금융 계산 엔진"

구현_기능:
  매물_검색_및_필터링:
    - PostgreSQL B-tree 인덱스 최적화 쿼리
    - 실시간 조건 매칭 (안전등급, 연식, 가격대)
    - 복합 WHERE 조건문 기반 필터링

  TCO_분석_엔진:
    - 구매가격 + 감가상각 + 유지비 + 보험료 + 연료비
    - EDA 탐색적 데이터 분석 (평균, 중위수, 분위수)
    - 3년/5년 비용 예측 (선형 추세 분석)

  금융_상품_매칭:
    - 할부/리스/일시불 최적 조합 계산
    - 월 납입금 계산기 (복리 공식: PMT 함수)
    - 신용등급별 금리 테이블 기반 매칭

  감가상각률_예측:
    - 브랜드별/차종별 역사적 감가율 평균값 활용
    - 연식/주행거리 기반 선형 회귀 분석
    - pandas DataFrame 기반 통계 계산
```

```python
class PracticalFinancialCalculator:
    def calculate_monthly_payment(self, principal, annual_rate, months):
        """표준 할부금 계산 공식 (PMT)"""
        if annual_rate == 0:  # 무이자 할부
            return principal / months

        monthly_rate = annual_rate / 12 / 100
        return principal * (monthly_rate * (1 + monthly_rate)**months) / ((1 + monthly_rate)**months - 1)

    def simple_depreciation_forecast(self, vehicle_data, years):
        """단순하지만 효과적인 감가상각 예측"""
        # pandas DataFrame에서 동일 브랜드/모델 평균 감가율 조회
        base_depreciation = {
            '현대': 0.15, '기아': 0.16, 'BMW': 0.12, '벤츠': 0.11,
            '제네시스': 0.18, '아우디': 0.13, '토요타': 0.14
        }

        brand_rate = base_depreciation.get(vehicle_data.brand, 0.15)
        age_penalty = vehicle_data.age * 0.02  # 연식 1년당 2% 추가 감가
        mileage_penalty = min(vehicle_data.mileage / 100000, 0.1)  # 주행거리 10만km당 10% 최대

        total_depreciation = (brand_rate + age_penalty + mileage_penalty) * years
        return vehicle_data.current_price * (1 - min(total_depreciation, 0.8))  # 최대 80% 감가
```
