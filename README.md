## 🚨 자동차 구매의 패러다임이 바뀝니다

### 💔 **현재 중고차 시장의 문제점**
- **정보 비대칭**: 딜러는 신뢰하기 어렵고, 구매자는 차알못
- **사기 위험**: 연간 사고차/침수차 피해 2,000억원
- **복잡한 선택**: 111,841개 매물 중 어떻게 고르나?
- **높은 컨설팅 비용**: 전문 상담 1회 30-50만원
- **시간 부족**: 직장인들의 딜러 방문 시간 제약

### 🎯 **CarFin AI가 해결하는 핵심 문제**

#### 🤖 **"AI 멀티 에이전트가 서로 협업하며 분석"**
> 차량전문가 + 금융전문가 + 리뷰분석가가
> 실제 데이터를 분석해서 맞춤형 차량을 추천해드립니다

#### 🛡️ **"데이터 기반 신뢰성"**
> 실제 매물 데이터와 실구매자 리뷰 분석으로 신뢰할 수 있는 정보 제공
> 분석 과정을 투명하게 공개하여 판단 근거 확인 가능

#### 💰 **"무료 + 간편 + 맞춤형"**
> 복잡한 중고차 정보를 AI가 쉽게 정리
> 24시간 언제든 빠른 분석과 개인 맞춤 추천

5. **💬 AI 상담**: 추가 질문과 맞춤 조언

### 🏆 **3명의 AI 전문가 협업 시스템**

#### 🚗 **차량 전문가 (Vehicle Expert)**
```yaml
분석 영역: 차량 사양, 성능, 안전성, 가성비 평가

```

#### 💰 **금융 전문가 (Finance Expert)**
```yaml
분석 영역: 가격 적정성, 할부 조건, 차량 가치 예측
AI 모델: LSTM+XGBoost 기반 가격 예측
핵심 기능: 총 비용 계산, 금융 옵션 비교
```

#### 📝 **리뷰 분석가 (Review Expert)**

### 📊 **빅데이터 + AI 파이프라인**

#### 🗃️ **데이터 레이어 (PostgreSQL AWS RDS)**
```yaml
규모: 111,841건 실제 중고차 매물 데이터
구성:
  - vehicles: 차량 기본정보 + 사양 + 가격
  - insurance_history: 보험개발원 연동 사고이력
  - detailed_options: 60개 세부 옵션별 가치평가
  - brands/models: 브랜드별 신뢰도 매트릭스
  - user_preferences: 개인화 학습 데이터

실시간성:
  - 매일 최신 시세 업데이트
  - 실시간 사고이력 체크
  - 동적 가격 변동 반영
```

#### 🧠 **AI 모델 레이어**
```yaml
하이브리드 추천 시스템 (차량 전문가):
  - 협업 필터링: 유사 사용자 선호도 분석
  - 콘텐츠 기반: 차량 사양 및 특성 매칭
  - 인기도 기반: 실제 구매 트렌드 반영
  - 브랜드 신뢰도 매트릭스 활용

KoBERT (리뷰 감정분석):
  - 한국어 리뷰 감정분석
  - 실제 구매자 만족도 추출
  - 차량별 장단점 키워드 분석

LSTM+XGBoost (가격예측):
  - 시계열 가격 변동 학습
  - 차량 가치 예측 모델
  - 시장 변수 기반 예측
```
