## 📊 **프로젝트 개요**

### **해결하려는 문제**
중고차 구매자들이 겪는 실제 불편함:
- 수많은 매물 중에서 내게 맞는 차를 찾기 어려움
- 차량 상태와 적정 가격을 판단하기 어려움
- 대출, 할부 등 금융 상품을 따로 알아봐야 하는 번거로움
- 전문 지식 없이는 현명한 선택이 어려움

### **CarFin AI의 해결책**
**3개 AI 에이전트**가 협력하여 해결:
1. **차량 전문가**: 성능, 가격, 상태를 분석해서 추천
2. **금융 전문가**: 개인 신용과 상황에 맞는 대출/할부 상품 추천
3. **종합 상담사**: 차량과 금융을 종합해서 최적의 선택지 제시

## 🎯 **핵심 가치**

### **개인화된 추천**
- 나이, 소득, 용도, 선호도를 고려한 맞춤 추천
- 85,320건의 실제 엔카 데이터에서 최적 매물 발굴
- 딥러닝 모델로 취향과 패턴 학습

### **전문가 수준 분석**
- AI가 차량 상태, 시세, 성능을 종합 분석
- 금융 상품 비교와 최적 조건 자동 계산
- 복잡한 정보를 쉽게 설명

### **원스톱 서비스**
- 차량 추천 → 금융 상품 매칭 → 구매 결정까지 한 번에
- 여러 사이트를 돌아다닐 필요 없이 통합 서비스

## 🚀 **기술적 특징**

### **현대적 기술 스택**
```yaml
AI/추천 엔진:
- PyTorch: 딥러닝 추천 모델 (NCF 알고리즘)
- Google Vertex AI: 최신 AI 기술 활용
- 실시간 학습: 사용자 피드백으로 지속 개선

백엔드:
- FastAPI: 빠른 응답속도의 Python API
- PostgreSQL: 안정적인 대용량 데이터 처리
- Google Cloud: 확장 가능한 클라우드 인프라

프론트엔드:
- Next.js: 모던 웹 프레임워크
- TypeScript: 안정적인 코드 작성
- 반응형 디자인: 모바일/PC 모두 지원
```

### **지능적 추천 시스템**
- **협업 필터링**: 나와 비슷한 사용자들의 선택 패턴 분석
- **콘텐츠 기반**: 차량 스펙과 사용자 요구사항 매칭
- **행동 분석**: 클릭, 관심 등록 등의 행동 패턴 학습
- **실시간 업데이트**: 새로운 매물이 등록되면 즉시 추천에 반영

### **3개 에이전트 협업 시스템**
```typescript
// 각 에이전트가 전문 분야별로 분석 후 결과를 종합
차량전문가.분석() + 금융전문가.분석() + 종합상담사.조율()
= 개인 맞춤 최적 솔루션
```

## 🏗️ **기술 아키텍처**

### **시스템 구성**
```
사용자 웹/앱
    ↓
Next.js 프론트엔드 (Vercel)
    ↓
FastAPI 백엔드 (Google Cloud Run)
    ↓
┌─────────────────┬─────────────────┐
│ 추천 AI 엔진    │ 금융 매칭 엔진  │
│ (PyTorch)       │ (Vertex AI)     │
└─────────────────┴─────────────────┘
    ↓
PostgreSQL 데이터베이스 (AWS RDS)
```

### **핵심 기술 선택 이유**
- **PyTorch**: 추천 시스템 연구에서 가장 많이 사용되는 프레임워크
- **FastAPI**: Python 중 가장 빠른 성능, 자동 문서화
- **Next.js**: React 기반 최신 웹 기술, SEO 최적화
- **Google Cloud**: AI/ML 서비스가 우수하고 확장성 좋음
- **PostgreSQL**: 복잡한 데이터 분석에 최적화된 안정적 DB
