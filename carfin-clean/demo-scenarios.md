## 🚀 시나리오 2: API 기술 데모

### curl 명령어 시연
```bash
curl -X POST http://localhost:3005/api/chat \
  -H "Content-Type: application/json" \
  -d '{
    "question": "골프백 들어가는 BMW 차량 추천해주세요",
    "questionCount": 1,
    "selectedPersona": "ceo_executive"
  }'
```
